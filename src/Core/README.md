# Alex75.AIAgents.Core

A .NET 10 F# library for AI Agents built with the Microsoft Agent Framework.

## Installation

```bash
dotnet add package Alex75.AIAgents.Core
```

## Usage

```fsharp
open Alex75.AIAgents.Core

let greeting = Library.hello "World"
printfn "%s" greeting
```

## Features

- Core agent abstractions
- F#-friendly API design
- Built on Microsoft Agent Framework
